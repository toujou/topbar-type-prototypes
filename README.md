# Topbar and navigation types prototypes
Interactive prototypes to explore different topbar and main navigation styles.

## Why
We would like to give our clients the option to select the topbar and main navigation types / styles for mobile and desktop devices in the TYPO3 backend.

This means we need to:
- identify useful topbar and main navigation types
- find a way to apply the correct styles and markup depending on the chosen types and the current viewport size
- find markup that is good, accessible, and highly customizable

## What's in here
These prototypes let us preview and compare different topbar and navigation styles side by side,
switch between them live,
and test specific combinations — including using a different style on mobile than on desktop —
to validate that each option holds up across devices before it becomes a real backend setting.
